# ◈ NEON RUNNER ◈

[![Python](https://img.shields.io/badge/Python-3.7+-blue?logo=python)](https://www.python.org/)
[![Tkinter](https://img.shields.io/badge/Tkinter-GUI-green)](https://docs.python.org/3/library/tkinter.html)
[![License](https://img.shields.io/badge/License-MIT-yellow)](LICENSE)

**Neon Runner** — это динамичный сайд-скроллер в стиле киберпанк, написанный на чистом Python с использованием встроенной библиотеки Tkinter. Без внешних зависимостей!

> Беги, прыгай, собирай монеты, уничтожай врагов и становись сильнее с помощью бонусов!

![Геймплей](https://via.placeholder.com/800x400?text=Neon+Runner+Screenshot) <!-- Замените на реальный скриншот -->

## ✨ Особенности

- 🌃 **Киберпанк-эстетика** — неон, частицы, пульсирующие цвета и звёзды.
- 🗺️ **Большой открытый мир** — 5000 пикселей для исследования.
- 🎮 **Разнообразные механики**:
    - 🪙 Монеты (обычные и бонусные)
    - 👾 Враги с разными типами поведения
    - 🧩 Бонусы (двойной прыжок, ускорение, доп. жизнь)
    - 🚀 Движущиеся платформы
    - 🔪 Острые шипы
- 🖥️ **Полностью интерфейс на Tkinter** — без сторонних библиотек.
- 🎨 **Анимированные частицы** — искры от прыжков, эффекты сбора предметов и гибели.
- 🗺️ **Мини-карта** для отслеживания прогресса.
- 💾 **Сохранение состояния** внутри сессии (счёт, жизни).

## 🎮 Управление

| Действие              | Клавиши                |
|-----------------------|------------------------|
| Движение влево        | `←` / `A`              |
| Движение вправо       | `→` / `D`              |
| Прыжок                | `ПРОБЕЛ` / `↑` / `W`   |
| Пауза                 | `ESC`                  |
| Выход в меню (после GAME OVER / WIN) | `ПРОБЕЛ` |

## 🔥 Бонусы и усиления

| Бонус          | Эффект                                | Внешний вид |
|----------------|---------------------------------------|-------------|
| 🟡 **Double Jump** | Разрешает один дополнительный прыжок в воздухе | Жёлтый прямоугольник |
| 🔵 **Speed Boost** | Увеличивает скорость передвижения x1.4 | Голубой прямоугольник |
| 💗 **Extra Life**  | Восстанавливает +1 жизнь (максимум 9) | Розовый прямоугольник |

## 👾 Типы врагов

- **Normal** — стандартный враг, красный.
- **Fast** — двигается быстрее.
- **Tough** — более крупный и опасный (фиолетовый).
- **Boss** — очень быстрый и опасный.
- **Slow** — медленный, но может зажать в углу.

## 🏆 Цель игры

Собрать **все монеты** на уровне, избегая врагов и шипов. Как только вы соберёте последнюю монету — уровень пройден! Также можно победить всех врагов для альтернативной концовки «ALL CLEAR!».
